# Chapter7 Compose

1. [ ] Install Podman Compose or Docker Compose.
2. [ ] Write a `docker-compose.yml` (or `podman-compose.yml`) with two services (web + db).
3. [ ] Start the application with `podman-compose up`.
4. [ ] Verify communication between services (web -> db).
5. [ ] Scale the web service to 3 replicas using Compose.
6. [ ] Stop the application with `podman-compose down`.
7. [ ] Add environment variables in the compose file.
8. [ ] Map ports from containers to host and test with curl/browser.
9. [ ] Mount a volume via compose file for data persistence.
10. [ ] Summarize best practices for multi-container apps with Compose.
